# Landing Page

A React landing page built with [Vite](https://vitejs.dev/).

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or newer recommended)

## Setup

```bash
npm install
```

## Development

Start the dev server with hot reload:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project structure

- `src/App.jsx` – Main landing page (hero, features, CTA, footer)
- `src/App.css` – Component styles
- `src/index.css` – Global styles and CSS variables
- `index.html` – Entry HTML
- `public/` – Static assets

Edit `src/App.jsx` and the CSS files to customize copy, sections, and styling. The design uses a dark theme with an accent color; you can change `:root` variables in `src/index.css` to adjust the look.
